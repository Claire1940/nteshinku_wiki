<Section icon={Sparkles} title="スキルの優先順位とステータスの進め方">

**NTE シンクの能力**を最大限に引き出すには、タレントのアップグレードを優先し、正しいステータスを装備する必要があります。変身中はキット全体にダメージが分散されるため、バランスの取れた投資が推奨されます。しかし、アルティメットがバーストローテーションの絶対的な核であることに変わりはありません。

### スキル強化の優先順位
1. **アルティメット：真紅の怒り**（変身バフと処刑ダメージの倍率を強化）
2. **通常スキル：高速突破**（機動力と変身中のスキルダメージを強化）
3. **通常攻撃：特殊武術**（基本のリソース生成能力を向上）
4. **サポートスキル**（ユーティリティを提供するが、純粋なダメージ倍率は低い）

### 推奨ステータスと装備

| 装備スロット / ステータス | 理想的な属性 | 優先度 |
| :--- | :--- | :--- |
| **プライマリカートリッジ** | 失われた輝き (Sクラス) | ★★★★★ |
| **メインステータス（カートリッジ）** | コスモスダメージ % | ★★★★★ |
| **サブステータス** | 会心率 / 会心ダメージ | ★★★★★ |
| **二次サブステータス** | 攻撃力 % / 全ダメージ % | ★★★★☆ |

<Section icon={Target} title="ステップバイステップ：シンクの戦闘ローテーション">

シンクのフィールド上でのDPSを最大化するには、通常形態と「真紅の奔流」状態の移行をマスターする必要があります。敵のシールドを粉砕し、大規模なコスモスダメージを与えるための最適な戦闘ローテーションに従ってください。

<Steps>
  <Step num="1" title="不屈の精神を溜める">
    通常形態で戦闘を開始します。5段の**特殊武術**コンボを実行します。3、4、または5段目の攻撃中に攻撃ボタンを長押しして、追加エネルギーを得るための変化攻撃をトリガーします。
  </Step>
  <Step num="2" title="高速突破を使用する">
    通常スキルの**高速突破**を使用して距離を詰めます。これにより多段コスモスダメージを与え、**不屈の精神**ゲージの残りを急速に満たします。
  </Step>
  <Step num="3" title="真紅の奔流を発動する">
    ゲージが完全に満タンになったら、アルティメットキーを押して**真紅の怒り**を発動します。彼女はすべての不屈の精神を消費し、強力な初期範囲ダメージを与えてから13秒間の変身状態に入ります。
  </Step>
  <Step num="4" title="変身コンボを実行する">
    変身中は、強化された通常攻撃である**タイダルアサルト**を連打します。強化スキル**天からの紅刃**は、クールタイムが明け次第使用して高いDPSを維持し、HPを全回復させます。
  </Step>
  <Step num="5" title="フィニッシャーを放つ">
    13秒間の変身時間が終了する直前に、アルティメットボタンを連続タップして**クリムゾンジャッジメント**を実行します。これにより高速ダッシュの処刑攻撃がトリガーされ、壊滅的な最後の一撃を与えます。
  </Step>
</Steps>

<Callout type="success" title="抑止の眼差しのメカニズム">
オープンワールドを探索中、シンクは非戦闘時または攻撃していない間、1秒ごとに**抑止の眼差し（Deterrent Gaze）**のスタックを獲得します。最大8スタックの状態で戦闘に入ることができ、最初の一撃で巨大な即時ダメージを与え、アルティメットエネルギーの一部を即座に返還します。
</Callout>
