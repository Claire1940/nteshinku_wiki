<Section icon={Sparkles} title="必須のNTE シンク素材＆周回リスト">

シンクのようなS級キャラクターをアップグレードするには、多様なリソースが必要です。突破素材からスキル強化アイテムまで、**NTE シンクの素材**チェックリストは、ハンターガイド、染料、通貨の3つの主要カテゴリーに分けられます。これらのアイテムは、新しいウォーレン大陸や「999夜」イベントショップに点在しています。

| 素材名 | タイプ | 主な入手先 |
| :--- | :--- | :--- |
| **新鋭ハンターガイド** | スキル強化 | デイリー依頼 / コード報酬 |
| **精鋭ハンターガイド** | スキル強化 | 999夜イベントショップ / 高レベルアノマリー |
| **明光の染料** | 突破素材 | ウォーレン大陸の探索 |
| **無色の染料** | 突破素材 | エイボン骨董店での交換 |
| **混沌の染料** | 突破素材 | 週替わりボス / イベント報酬 |
| **カブトムシコイン** | イベント通貨 | 999夜ミニゲーム |
| **フォン** | 基本通貨 | 通常プレイ / クエスト |

Ver.1.2の期間中にこれらの素材を最も効率的に集める方法は、期間限定イベント「999夜」を通じることです。イベントショップ（ボタンショップ）では、カブトムシコインを消費して、通常では大量入手が困難な上位の「混沌の染料」や「精鋭ハンターガイド」と交換できます。

<Callout type="info" title="リソース管理">
**DRAGONGIRL** や **DREAMWALK0603** といった有効なシリアルコードを使用して、フォンや新鋭ハンターガイドを事前に入手しておくのを忘れないでください。
</Callout>

「羞花の幻影」を入手するには、アークショップへ移動し、「**覚悟の特別アーク研究プログラム**」に参加する必要があります。このプログラムはシンクのキャラクターガチャと並行して開催されます。ここではトライキーまたは覚悟のキーの使用が必須です。キーが不足している場合は、シンクの基本性能がA級アークでも十分に強力であるため、まずはキャラクターの獲得を優先しましょう。

<Callout type="tip" title="プロのアドバイス">
研究中に「カメリア結社」や「荒れ狂う火炎」などの他のS級アークを入手した場合は、保管しておきましょう。これらはシンクの最適装備ではありませんが、同じガチャに含まれるミントやエドガーといったA級キャラクターには非常に適しています。
</Callout>

<Section icon={Target} title="ステップバイステップ：事前準備＆育成ガイド">

シンクを入手した瞬間に戦力として活用できるよう、以下の構造化された周回ルートに従ってください。このガイドでは、Ver.1.2の最初の1週間で効率を最大化することに焦点を当てています。

<Steps>
  <Step num="1" title="ウォーレン大陸を解放する">
    ストーリーエピソード「龍との戦い」を完了します。これにはハンターレベル26とクエスト「夢歩きの回廊」の完了が必要です。このエリアには、シンクの突破に必要な染料のドロップポイントがあります。
  </Step>
  <Step num="2" title="イベント通貨を最大化する">
    毎日「999夜」のミニゲームをプレイしましょう。ボタンショップでカブトムシコインを使い、コスチュームに手を出したくなる前に「精鋭ハンターガイド」と「混沌の染料」の在庫をすべて買い占めてください。
  </Step>
  <Step num="3" title="有効なコードをすべて引き換える">
    **RACENOLIMIT** や **TOMATO100** などのコードを入力して、250個以上のアニュリスと10,000以上のフォンを確保しましょう。これらのリソースは、キャラクター昇級の最終段階で不可欠になります。
  </Step>
  <Step num="4" title="シンクの戦闘チュートリアルを完了する">
    Ver.1.2にはシンク専用のチュートリアルが追加されています。これを完了することで、彼女の攻撃タイミングや「龍娘」としてのメカニクスを学べるだけでなく、基本的な強化素材も報酬として得られます。
  </Step>
  <Step num="5" title="絆レベルを上げる">
    シンクを入手したら、都市での交流やホームタイムを行いましょう。絆レベル10に到達すると、ショップ販売スキンに代わる無料の衣装「**迷宮の百合**」が解放されます。
  </Step>
</Steps>

<Callout type="warning" title="期間限定の報酬">
ショップ衣装「ドラゴンハンター」や「暁の巡回」の割引期間は2026年8月19日に終了します。リフトクリスタルで購入を検討している場合は、最初の1ヶ月以内に購入すれば20%お得になります。
</Callout>

</Section>
