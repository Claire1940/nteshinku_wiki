<Section icon={Target} title="NTEシンク見送り：判断基準マトリクス">

シンクを引くか、あるいは**NTEシンクを見送る**決断をするかは、『無限の彼方（NTE）』Ver.1.2「999夜」を迎えるプレイヤーにとって最も重要な問いです。このパッチ最初のS級キャラクターとして登場するシンクは、非常に派手で攻撃的なコスモス属性の近接格闘スタイルを持っています。しかし、フェーズ2には待望のアニマ属性ヒーラー兼バッファーである「イロイ」の登場が控えており、限られたガチャリソースの管理は極めて重要です。

引くべきか見送るべきかの判断を助けるため、シンクを獲得することで得られるメリットと、見送ることで節約できるリソースを比較しました。

| シンクを引くメリット | シンクを見送るメリット |
| :--- | :--- |
| **トップクラスのコスモス属性メインDPS**として、高い瞬間火力を発揮できる。 | フェーズ2や将来のバナーのために、貴重な**三和キーやアニュリスを温存**できる。 |
| **独自のダッシュメカニクス**により、オープンワールドでの高速移動が可能になる。 | メインDPSの枠が既に埋まっている場合、**リソースの枯渇を回避**できる。 |
| ガチャの回数報酬で**限定コスメティック**を入手できる。 | 多くのチームに編成可能な**イロイのようなサポートユニット**に集中できる。 |

<Callout type="tip" title="プロのアドバイス">

<Section icon={Swords} title="シンクの戦闘メカニクスとプレイスタイルの分析">

シンクがあなたのチームに適しているかを理解するには、彼女独自の2形態戦闘ループを分析する必要があります。シンクは標準的なキャラクターのように究極エネルギーを生成するのではなく、特殊なゲージシステムで動作します。

### 2形態ループ：通常時 vs サージング・クリムゾン
通常状態のシンクは、素早い近接格闘攻撃を使用して**「不屈の闘気ゲージ」**を溜めます。このゲージが最大になると、究極技『クリムゾン・フューリー』を発動してゲージを消費し、伝説的な**「サージング・クリムゾン（昂揚状態）」**へと移行します。

```
[通常形態：基本攻撃] ➔ [不屈の闘気を溜める] ➔ [究極技発動：クリムゾン・フューリー] ➔ [サージング・クリムゾン形態へ]
```

サージング・クリムゾン状態では、すべてのスキルが強化され、高倍率のバリエーションに置き換わります。広範囲への攻撃（AoE）能力、攻撃速度の上昇、そして高い中断耐性を獲得します。

### 「威圧の眼差し」メカニクス
能動的な攻撃以外にも、シンクは戦闘中に2秒間攻撃を控えることで**「威圧の眼差し（Deterrent Gaze）」**スタックを蓄積します。最大8スタックまで保持でき、次の攻撃時に全スタックを消費して**「インスタント・ストライク」**を放ちます。この攻撃は甚大なダメージを与え、究極技エネルギー、スキルエネルギー、サイクルメーターエネルギーを即座に回復します。

### オープンワールドでの利便性
オープンワールドでは、シンクは回避ボタンを長押しすることで「ランページ・スプリント」を発動できます。これにより、車両を召喚することなくヘセロウの街を驚異的なスピードで駆け抜けることが可能です。

<Callout type="warning" title="プレイ上の注意点">
シンクの眼差しスタックは、消費する前にキャラクターを交代させると完全に消失します。初期の覚醒（凸）ノードを進めることで、控えに回ってもスタックを維持できるようになり、この不便さが解消されます。この覚醒がない場合、頻繁にキャラを切り替えるプレイスタイルではローテーションが制限されると感じるかもしれません。
</Callout>

</Section>

<Section icon={Shield} title="ガチャ価値：『夜明け前』の詳細">

限定ボード『夜明け前（Before the Dawn）』では、シンクと共にいくつかの貴重な報酬やピックアップキャラクターが登場します。これらのマイルストーンを理解することが、ガチャ計画を立てる鍵となります。

### ガチャスケジュールと条件
- **期間**: 2026年7月8日（メンテナンス後）〜2026年7月29日 05:59 UTC+8。
- **解放条件**: プロローグ「新たな目的地 - エイボン・アンティークショップへ向かう」をクリア。
- **ピックアップA級キャラクター**: ミント、エドガー、アドラー。

### ガチャ回数報酬（マイルストーン）
標準的なガチャとは異なり、『夜明け前』では特定の回数に達すると限定のコスメティックアイテムを受け取ることができます：

| ガチャ回数 | 報酬タイプ | 報酬名と詳細 |
| :--- | :--- | :--- |
| **50連** | グライダー外装 | **オーバーキャスト・キャノピー**: スタイリッシュなカスタムグライダースキン。 |
| **120連** | 車両外装 | **隠れ龍（Hidden Dragon）**: 車両「Blizzard-V4」用のプレミアム外装。 |
| **200連** | キャラクター衣装 | **テラシーの学生（Student of Terrasea）**: シンク専用のアカデミックなテーマのスキン。 |

<Callout type="info" title="コスメティックに関する注記">
これらのマイルストーン報酬は、確定報酬に達する前でもガチャを引いている最中にランダムでドロップする可能性があります。
</Callout>

</Section>

<Section icon={Sparkles} title="ビルドの基本とモチーフアークへの投資">

シンクを引くことを決めた場合、彼女のダメージを最大化するには「アーク（Arc）」と呼ばれる武器への投資が必要です。シンクは凝縮（Condensate）タイプのアークを使用し、彼女にとっての最適（Best-in-Slot）オプションは、限定S級アークの**「ブラッシング・ミラージュ（Blushing Mirage）」**です。

### モチーフアーク：ブラッシング・ミラージュ
- **入手方法**: 決意の限定アーク研究プログラム（シンクのガチャと並行して開催）。
- **使用通貨**: 三和キーまたは決意キー。
- **主要効果**: クリティカルヒットを当てるたびに会心ダメージが4%増加し、最大10スタックで計40%の会心ダメージボーナスを獲得します。
