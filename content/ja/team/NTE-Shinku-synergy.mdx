### チャージ反応の仕組み

チャージ反応は、『Neverness to Everness』における多段階の元素プロセスです。これを発動するには、チームで**レモラ**と**ブロッサム**の両方の反応を誘発する必要があります。

| 反応タイプ | 元素の組み合わせ | 敵への戦闘効果 |
| :--- | :--- | :--- |
| **レモラ** | コスモス + ラクシャナ | 持続的な印を付与し、ターゲットに対するチームの会心率をアップする |
| **ブロッサム** | コスモス + アニマ | 範囲ダメージを与え、ターゲットの元素耐性を低下させる |
| **チャージ** | レモラ + ブロッサム | チーム全体の必殺技エネルギーを大幅に回復し、範囲爆発を引き起こす |

シンク（コスモス）、ラクシャナキャラクター、アニマキャラクターを含むチームを編成することで、自己完結型のエネルギー回復ループが確立されます。チャージ反応が発生するたびに、シンクは必殺技エネルギーを大量に獲得し、最小限のダウンタイムでサージングクリムゾン状態に戻ることができます。

<Callout type="tip" title="シナジーのコツ">
シンクのパッシブスキルは、チャージ反応をチーム全体のエネルギー生成へと変換します。これにより、彼女は素早いサイクルやクイックスワップ（即時交代）主体のパーティー編成において非常に優秀なドライバーとなります。
</Callout>

### 1. 最強メタ編成：必殺技チャージコア
この編成はシンクの絶対的なベストインスロット（最適）チームであり、最大のダメージ増幅、集敵、および防御的なユーティリティを提供します。

| キャラクター | 元素 / 役割 | 主なシナジー効果 |
| :--- | :--- | :--- |
| **シンク** | コスモス / メインDPS | オンフィールドのアタッカー。チャージ反応を純粋な爆発ダメージに変換する |
| **ハトル** | ラクシャナ / サブDPS | レモラを誘発し、印の持続時間を延長、さらに会心率を10%上昇させる |
| **イロイ** | アニマ / サポート | HP依存の回復、汎用ダメージバフを提供し、会心ダメージを強化する |
| **ジウユエン** | アニマ / サブDPS | クラウドコントロールで敵を集め、アニマ耐性を低下させる |

### 2. ダブルコスモス・サイクル編成
ローテーション速度を最大化し、即座に反応を誘発したいプレイヤーには、シンクと2人目のコスモスユニットをペアにすることが非常に効果的です。

| キャラクター | 元素 / 役割 | 主なシナジー効果 |
| :--- | :--- | :--- |
| **シンク** | コスモス / メインDPS | 強化変身中のメインアタッカー |
| **ゼロ** | コスモス / サポート | 超能力サイクルゲージを素早く回復し、エネルギー獲得時にチームを回復する |
| **ハトル** | ラクシャナ / サブDPS | チャージ反応を可能にし、安定したオフフィールドの爆発力を提供する |
| **ナナリー** | アニマ / サブDPS | 超能力サイクルダメージが発生するたびに、追撃のアニマ攻撃を発動する |

### 3. 無課金向けおすすめ編成
ガチャ石を節約している場合や、限定Sクラスのサポートを逃してしまった場合でも、この非常に入手しやすい編成で核心となるシナジーのループを十分に発動させることができます。

| キャラクター | 元素 / 役割 | 主なシナジー効果 |
| :--- | :--- | :--- |
| **シンク** | コスモス / メインDPS | 「サベージコットン」などの無課金アークを活用できる、入手しやすいSクラスのアタッカー |
| **ハトル** | ラクシャナ / サブDPS | バージョン1.2のメールで配布される無料のSランク選択ボックスから入手可能 |
| **エドガー** | コスモス / サポート | シンクのガチャでピックアップ。追加のコスモス粒子と回復を提供する |

  </Step>
  <Step num="4" title="集敵とデバフ">

  <Step num="5" title="紅蓮昇華の解放">

  </Step>
</Steps>
